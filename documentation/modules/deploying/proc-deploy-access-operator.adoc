= Deploying the Access Operator

// This is a temporary note that is expected to be removed after Access Operator 0.3.0 release.
NOTE: The Strimzi Access Operator version 0.2.0 does not support the Strimzi `v1` CRD API and is not compatible with Strimzi {ProductVersion}.
The Strimzi `v1` CRD API will be supported in Strimzi Access Operator 0.3.0 release that will follow the Strimzi {ProductVersion} release.
Please check the link:https://github.com/strimzi/kafka-access-operator[Access Operator repository^] for the latest updates on the Access Operator releases and compatibility with Strimzi {ProductVersion}.
